# vast-analyze-tool

> A tool to manage VAST analyze results conveniently

## Build Setup

``` bash
# install dependencies
npm install
# or
yarn

# serve with hot reload at localhost:9080
npm run dev
# or
yarn dev

# build electron application for production
npm run build
# or
yarn build
```
